Kanban Project Management Tool

What is Kanban?

Kanban is a visual system for managing work as it moves through a process.
Kanban visualizes both the process (the workflow) and the actual work passing through that process.
The goal of Kanban is to identify potential bottlenecks in your process and fix them so work can flow through it cost-effectively at an optimal speed or throughput.

Summary

Here you can find an overview about amount of tasks, how many tasks which status have and when is next urgent coming up task.

Add Task

Create new tasks with following parameters:

  • Title: give decisive title
  • Description: add description to explain details of task
  • Category: choose category of department or create new category
  • Assigned to: choose members to assign the task to
  • Due date: pick latest date when task should be finished
  • Prioriy: Choose kind of priority of urgency
  • Subtasks: Add new subtasks, so you can break the task down into multiple stages of completion

After creating task you can find it in the board.

Board

All created tasks will show in board. Drag tasks between the four sections "To Do", "In progress", "Awaiting feedback" and "Done"
depending on current working status. Tasks, which includes subtasks have a progressbar in addition, which show you, how many subtasks are done.
You can click on task to open an detail overview, which can be edited as well. It's also possible to add new tasks from this page.
Find tasks with search function close to headline. You can also search for tasks by assigned members and priority.

Contacts

Here you find your contacts. View Contact information, edit your already existing contacts or create new contacts.
